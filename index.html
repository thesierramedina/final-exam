Getting Started

Getting started with website building can be an easy and simple process! Get started by creating an Amazon Ligthsail account on Amazon Web Services. After your account is created, you are going to create an instance. Check out this video for some more information on getting started with Amazon Lightsail!

Connecting

After you create an instance using a tool of your choice (WordPress, Joomla, LAMP PHP, etc.) you are going to connect to your own computer by using the connect using SSH button. Here is where you can configure your MySQL databases using MariaDB.

Make sure that whatever tool you choose to use has been downloaded onto your computer. You are going to need a special password key in order to access your databases, you can create that on AWS Lightsail.

Next Steps

I prefer to utilize Bootstrap or from-scratch coding because it allows much more creative license, but since you're here WordPress must be your jam so you'll also need cyberduck, a software that helps you add plugins and use various templates as well as connect files.

Wrapping Up

Make sure that cyberduck is connected to your FTP so that all your files can be connected to the cool databases we've set up.

Enjoy Your Site!

All that's left to do it get into WordPress and start messing around with your pages and posts, this is where you can add content and customize the templates to best fit your content and style. Have fun!
